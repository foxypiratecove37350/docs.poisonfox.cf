gougle_G# History

First question

gougle_G# was created by Karl LÉOGNANY and ChatGPT the 26/02/2023 with one question : "How to create a programming language ?". google_G# was created with a chat enter Karl LÉOGNANY and ChatGPT, the GSDev.

Libraries

gougle_G# has a wide variety of libraries to help you build your apps. Here is a list of core libraries :

  • GSystem : system library for managing files, directories, processes, etc.
  • GSDisplay : library for creating graphical user interfaces (GUI).
  • GSNetwork : library for network communication.
  • GSDatabase : library for database management.

Libraries were created by GSDev, GSDatabase and GSNetwork are created just by ChatGPT.
